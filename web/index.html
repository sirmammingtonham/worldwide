Layers of a website

HTML

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of a website. Other technologies besides HTML are generally used to describe a webpage's appearance (CSS) or functionality (JavaScript). "HyperTexts" are links that connect webpages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By linking your content to pages created by other people, you become an active participant in the World Wide Web.

Like everything on the internet, languages evolve over time. Web languages need regular upgrades in order to stay current and solve new problems faced by web developers. HTML5 is the latest version of HTML.

CSS

CSS (short for Cascading Style Sheets) is a style sheet language used to describe the style of a document concerning colors, typography, placement and more. CSS is one of the core languages of the open Web and is designed to enable the separation of presentation and content. This separation can improve content accessibility, provide more flexibility and control, and enable multiple web pages to share formatting by specifying the relevant CSS in a separate .css file. This website has over 6,000 lines of CSS that define its modern look and feel.

Developed in levels, CSS1 is now obsolete, CSS2.1 is a recommendation, and CSS3 is the most current iteration, though not adopted by some browsers.

JavaScript

JavaScript (JS) has been popularized as a client-side scripting language for websites. JavaScript has become the de-facto language of choice when building websites, and one of the most popular ones to build web applications. JS is commonly used to create highly responsive interfaces that improve the user experience and provide dynamic functionality, without having to wait for the server to react and show another page.

PHP

PHP (Hypertext Preprocessor), is a server-side scripting language that often gets paired with JavaScript. PHP is still in extensive use today for its ease of use, availability, and compatibility (it still powers the majority of servers). PHP can be used to collect form data, send and receive cookies, and connect to a company's database, among other things.

Python

Python is famous for its variety of use cases and simple programming typing style. Python is also used in many computer science classes, making the language very well known throughout the workforce as students enter the field. Python also has a popular web framework (Django) which can be used to generate dynamic webpages.

Many More

Ruby, Golang, ASP.NET, C#, Java, and more are also frequently used to create websites. In addition to the languages themselves, there are many different frameworks (Ember, React, Ruby on Rails, Django), databases (MySQL, MongoDB, GraphQL), and platforms (WordPress, Shopify) that facilitate the creation of increasingly complex websites.

Experience Modern Web Design

Minimalist design

Less is best

Minimal design in websites is essential for helping to create focus and clarity for your site users. It helps users to stay focused on your content and makes for simple navigation around your site.

Excessive javascript

Javascript is very useful for creating interactive websites, and is responsible for all of the animations on this site. However, many modern websites take it too far, and request all sorts of things from users that delay us from accessing their actual content.

Mobile friendly websites

Responsive design

60% of web traffic takes place through mobile devices! Many modern websites are optimized for use on many different devices.

Hero images and banners

Look at the top of this site

Hero images are essentially large banners, usually visible above the fold. Large images can help visually tell the story of the website without having to rely on just text. This trend is on the rise as well.

Web-safe typography

Big, clean fonts

Clean design typography will include the following elements: web-standard fonts, correctly sized text (usually bigger than 16px), adequate line-spacing for improved readability, limited use of different colored fonts, and grey or black typography, depending on the background tone/image.

Ghost and floating buttons

Look at the top right of this site, and the top.

Transparent buttons look great, and floating buttons provide easy navigation wherever you are in a site.
